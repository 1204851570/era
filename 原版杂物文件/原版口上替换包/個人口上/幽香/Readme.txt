・弾幕勝負とは？

『互いにBを振り、出た目の大きさを競うというのが基本のルールです。
　最初は互いに残機を3つ所持していて、戦闘能力が大きいほどBの数と面が増えます。
　プレイヤーのみ、自分の出目を見た後、专心回避か勝負するかを選ぶことができます（このとき、相手の出目は見えません）。
　专心回避ことを選んだ場合は、残機が減らない代わりに、出目の差に応じてHPにダメージを受けます。
　HPが0になると、プレイヤーの残機が1減ります。残機が0になれば、プレイヤーの敗北です。
　勝負することを選び、相手より小さい出目だった場合は、無条件でプレイヤーの残機が1減ります。
　勝負することを選び、相手より大きい出目を出したときに限り、相手の残機を1減らすことが出来ます。
　プレイヤーの残機が0になる前に、相手の残機を0にすることができれば、プレイヤーの勝ちとなります。
　また、プレイヤーはボムを使ってBの出目をプラスできるので、ここぞという場面で使いましょう。
　ただし、残機が減るとボム使用回数が1になるので、使わないまま残機を減らすのは損になります。』

- 要するに戦闘能力を高めてからサイコロ転がすだけです。
　戦闘能力はあんまり使い道が無さそうなのでこの機会に鍛えてみてください。
　ついでに恋慕or恋人の幽香と戦闘訓練するとより戦闘経験が上がるようにしてみました
　
- 残機を2つ以上減らせないまま負けると、ペナルティとしてお金か筹码の半分を粉砕されます（幽香と恋慕or恋人の場合はペナ無し）。
　ショタか女你だとペナルティを受けませんが、代わりに犯されます。
　ショタの条件は（年齢が兒童or幼稚）and（矮小体型or幼児体型）です。よく分からない人は幼稚・矮小体型持ちの男你を作ってください。
　女你で幽香が扶她だと別の分岐があります
　
- 勝つと幽香さんが何でも言うこと聞いてくれるらしいです。

以下攻略のヒント

・勝てねえんだけど

- ノーカラテ・ノーアナタ。備えよう。
　相手の戦闘能力-1の戦闘能力が最低レベルで、それで戦うのを想定した蔷薇ンスにしてます。それ未満で勝つのは多分不行。
　相手残機1になるとかなり強化されます。
- プレイヤーのB面は相手に比べて蔷薇つきが大きいので、いい値になるまで回避に専念した方がいいでしょう。
- 最大値・最小値はあんまり意味が無いので、大きいB値が出たと思ったら勝負したほうが勝ちやすいでしょう。
- 結局は運ゲーなので毎日挑戦してください。
　幽香さんはちょろいので戦うより普通に落としたほうが早いというのはあります
