菫子口上、TW　4.622
出会いから细心的、プラトニックに関係を進め、恋慕がついてから告白ことを想定しています。
…でも恋慕まで持っていくのは面倒くさい方のために、ウフフ口上は恋人になれば解禁されるようにしました。
你は男、童貞。
菫子は処女設定が一応前提になっています。

第３弾です。
やっぱりキャラ自体に自分の妄想が多分に盛り込まれているので、『こんなの俺の好きな会長じゃない！』と思う方はごめんなさい…
というか你に勝手な設定付け足してたり、恋人のバカップルぶりとか過去最高クラスに妄想入ってます。

キャラのコンセプトとしましては、出会った当初は気難しいうえに高自尊心。
でも、仲良くなると夢でも現実でもイチャイチャラブラブ、将来の誓いまでしちゃう典型的傲嬌という感じです。
デレるのを期待して、地道に付き合ってあげてください。
また、リビドーの赴くままに書き進めた結果、ところどころネタに走っております。
予めご了承ください。

また、この口上だけの完全オリジナル設定として、你は夢の中で幻想郷を訪れていて、現実では会長と同じ高校に通っているという背景があります。
他のキャラ口上とは噛み合わない部分が出てきますが、何卒ご了承ください。

現在、ごく一部の指令のみの実装となっています。
他の指令もぼちぼち書く予定ですが、お出しできるのがいつになるかはわかりません…
他の口上と並行する形にもなりますので、お待たせする可能性大です。

また注意していただきたい点として



※※※ この口上は実際のプレイを通しての動作確認は行っておりません ※※※



一応開発者向け設定での起動確認でエラーは出ませんでしたので公開に踏み切りましたが、プレイしている中でおかしいところが見つかるかもしれません。
そういったときはスレ等で遠慮なくご指摘をお願いします。頑張って直します。
