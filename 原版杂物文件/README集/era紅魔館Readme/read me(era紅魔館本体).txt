玩耍方
	1.外付け型オープニングを開いてオープニングを作成する
	2.作成したオープニングをじっくりと読む
	3.おもむろに同梱のEMUERAを起動する

バリアントの特徴
	・紅魔館を舞台としたADVゲームを目指してみた
		館見取り図を拡張
	・ベースはeracustom
	・MASTERのカスタムが可能
		男、女、扶她、ショタ、ロリお好きな属性でどうぞ
		男の娘、男装少女にも対応
	・TARGETが常に変動する
		MASTERと自動でうろうろする住人が同じ部屋にいるときにTARGET:1～が設定され、TARGET:1の住人がTARGETとなる
		同じ部屋に複数の住人がいるときはCFLAG:310により設定される館内での地位の順でTARGET:1～が設定される
		処理の都合上TARGETの初期値は0
	・住人と会話したり身体接觸したり、部屋の掃除をしたりできる
		TARGETとの親密が上がると指令ごとに積極的に行動してくるようになる
			counterは今のところSOURCE等の処理は行われず状況描写だけ
		理性と情緒、刻印、欲望、欲情、好意なんかも考慮するように
		着衣性騷擾を追加
			着衣状態での露骨な性騷擾をされます
	・能力の上昇は自動で行われる
	・現在的ところ館を清潔に保つことでMONEYが得られる。
		MONEYの使い道は道具の購入の他、館内での地位の上昇などを予定している
		部屋に住人がいるとそ的房間が時間経過により汚れていく
		清掃技能を持つ住人なら勝手に掃除してくれる
	・各種技能は経験により上昇する
	・時間
		TIMEで管理している時間は分刻みで、指令ごとに一定の時間が経過し、1440分経過すると１日繰り上がる
		茶を入れるなど特定の指令は一定時間経たないと実行できないようになっている
	・女仆妖精詰所
		住人の居場所、部屋の清潔度を調べることができる
	・日替わりパンツシステム
		よごれている部屋を掃除と脱ぎっぱなしのパンツをくすねることが出来るように
		パンツはSHOP画面で閲覧、売却(同種のパンツを2枚以上持っているとき一律100$)が可能
	・感情ゲージ
		情緒、理性、憤怒の三種類
		取得SOURCE、時間経過で上下する
		憤怒ゲージが限界に達すると憤怒状態になり、指令が制限され、また憤怒ゲージが減少しにくくなる
		情緒、理性はMASTERとの亲热強度、取得SOURCEに影響する
	・诶嘿嘿
		MASTERの亲热強度1以上で性騷擾が、2以上で推倒ことが可能になる
		脱衣服[820]で全裸にする
			脱衣はもっと充実される予定
		诶嘿嘿モードは 逢引終了[999] で解除できる
		诶嘿嘿中に3人目が同室すると诶嘿嘿モードは解除される
		诶嘿嘿中TARGETの体力が500を切ると強制解除され、TARGETはしばらく自室で就寝する
		二人以上と同室しているとき両者の亲热度が高いとまとめて推倒ことが出来るように
		性交系のソースは入るが振動棒などのソースはまだ入らない
	・反発刻印
		情緒低い、理性高い状態では减少SOURCEに補正がかかり反発刻印が付きやすい
		反発刻印がつくと数日間心情不快になるが日数経過で心情不快は解消し、反発刻印は除去される
			反発1では次の日には心情が直り反発刻印は除去されるが、2,3は数日かかる
		心情不快状態では常に憤怒ゲージがMAX
			指令をほとんど実行できなくなる
			诶嘿嘿中に反発刻印がつくと诶嘿嘿フラグが解除される
		反発刻印は何度でも取得するが反発刻印によるABL低下は一回だけ
	・要去了
		射精部位選択の代わりとして導入
		イキそうになるまでは射精しない
		勃起度を導入
			非勃起状態なら一部の指令が制限される
	・指令の整備
		指令を実行すると@COM999(ダミー)がよばれ@SOURCE_CHECKにより@COMXを呼び出す
		実行判定を@CAN_COMXに分離した
		指令ごとに体位がTEQUIP:101～に設定され、時間をさかのぼって何をしていたかがわかる
			舐陰→肛門愛撫では舐陰状態は引き継がれるが、舐陰→接吻ならば引き継がれない
				TEQUIP:101～はTEQUIP:111～に1ターン引き継がれる
				乳交と口交は本来排他的だが、TEQUIP:111～をみることで乳交と口交を連続的に行なったことを判定できる
			舐陰→肛門愛撫→６９(口交)というようなことが出来る
			TARGETごとに設定されるのでうまく動かせば3P,4P,～にも対応できるはず
				とりあえず雙重口交を実装
			バグ取り用に@SHOW_STATUSで今何をやっているか簡単な表示が出力される
				鬱陶しいと思う人はINFO.ERB 3,4行目の
					SIF TFLAG:102
						CALL SHOW_TOUCH
				を削除するかコメントアウトしてください
		派生指令はSELECTCOMでなくTFLAG:50で管理する
			SELECTCOMに干渉しないのでむやみに派生を増やしてテンプレを圧迫することもなくなる
			派生指令は、@COMABLE501～,@SCOM1～,@CAN_COM1～で管理する
			
		COMFX.ERBからPRINT系の命令を排除
			@COMX内でCALL COMXみたいな使い方が出来るようになった
				例：手淫口交(@SCOM5)ではまず口交を呼び出して手淫経験、快感強度を上書きしている
	・SOURCEの整備
		CUPを用いてキャラごとに管理するように
		TARGETが省略されていた部分は別関数にしてARGを用いるように
		MASTERとTARGETに設定されているキャラについてはSOURCE計算が回るように
	・陥落素質
		好感度1000以上、奉仕快楽経験200以上で恋慕取得
		恋慕取得時に相手的房間の合鍵をもらえる
		芙兰ちゃんの場合は幽閉が解かれる
		恋慕の相手的房間をMASTERの私室にしてそこで寝起きすることが出来る
		※NEW※
		各種の性技の熟練度が上がるように
		熟練度MAXで部分陥落素質もしくは技能素質を習得
		counter行動に影響
	・大浴場を設置
		時間経過により浴室への移動傾向が高くなる
		大浴場で诶嘿嘿状態になると浴室PLAYがつく
		それに伴い通常指令の浴室PLAYを廃止
		性騷擾、逆性騷擾するとガチエロになるので整備されるまでは会話と身体接觸程度、推倒、帯出去しか出来ない
		芙兰ちゃんおふろではおしりのあなもきれいにするんだよ诶嘿嘿
			浴室にはいると汚れがリセットされ、肛門的污垢が落ちる
			時間経過で肛門にアナル汚れがつく
			汚れに腸内も追加、肛门性交、肛門愛撫などでアナル汚れが肛門に付着
			汚臭無視がなくても舐肛が出来るようになった
構想中の仕様
	・性騷擾
		着衣日常系の指令を増やしたい
		感情ゲージ的なもので許容量を設定する
	・逆性騷擾
		着衣逆性騷擾を実装
		SOURCEがはいるように
			MASTERのSOURCEはTARGET側で管理
		脱衣性騷擾をいくつか実装
			もう稍微調整が必要
	・ウフフモード
		性騷擾を繰り返す、逆性騷擾で被逆推ると発生するように
		他の住人に見つかることがある
		仓库では見つからない
	・姉妹丼主従丼洋風中華丼
		PLAYRE-TARGET主体ではなくMASTER-TARGET:1 MASTER-TARGET:2…のようなシステムを目指す
	・侵入者
		たまに魔理沙が館に侵入しますが試験運用中です。特に害はないので生ぬるい目で見守ってください
	・テンプレ
		最大限詰め込むとテンプレサイズが膨大になり見ただけでやる気が奪われる気がする
			分割テンプレにより実質的には問題ないが全部埋めないといけないと思ってしまう作者がいそうな気がする
		テンプレは最小限のサイズに抑える
		小分岐集を別に作り、口上作者はそれを参考に分岐を作成する
		乳夾口交口上を書かなければ乳交口上が出るようなギミックを仕込む
		※NEW※
		テンプレにライセンス的なものをつけた
			基本的に作者の意思が不明な場合は加筆改変は可能ということにします
			改変元を明らかにするなど最低限のマナーはここに明記しなくても分かっているはず
