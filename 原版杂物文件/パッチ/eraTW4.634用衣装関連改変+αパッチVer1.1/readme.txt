eraTW4.634用衣装関連改変+αパッチ

このパッチはeraTW4.634用脱衣描写修正パッチとeraTW4.634用弾幕勝負勝利表記パッチに上書きして使用してください

衣装関連他で以下のことを改変しました
・Look.ERBの服の表記優先順位を変更
	袴を履いていても下半身の表記が和服だったなど表記が不自然だと思う場面があったので
	下半身は、
	裙子→和服→連衣裙→下半身外衣→下着
	上半身は、
	上半身外衣１→上半身外衣２→和服→連衣裙→下着と変更

・服のEQUIP使用番号をを変更
	上記の表記順にするために以下のように並べ替え
	EQUIP:13 裙子	→	EQUIP:13 連衣裙	
	EQUIP:14 上半身外衣１	→	EQUIP:14 和服
	EQUIP:15 上半身外衣２	→	EQUIP:15 裙子
	EQUIP:16 連衣裙	→	EQUIP:16 上半身外衣２
	EQUIP:17 和服		→	EQUIP:17 上半身外衣１

・PRINT_STATE.ERBの服装の表記順を変更
	付属を上げて襪子と靴を一番下へ

・服装追加と一部キャラの改変
	連衣裙に新規の服、肌襦袢を追加。和服を着ているキャラにある程度追加
	小鈴の連衣裙を肌襦袢に
	華扇の旗袍を導師服に
	その他もろもろ

・洋室の室内で靴を履くように

・室内なら飾品の傘を取るように

・脱衣描写一部修正

・TRACHECK_ABLUP.ERBの能力取得が何回も表記されると思われる記述を修正

・TRACHECK_ORGASM.ERBの複数絶頂の表記を整理

・EQUIPで番号を使っていた記述を文字列に変換


これらの改変に脱衣描写がまだ追いついていません。ご了承ください
